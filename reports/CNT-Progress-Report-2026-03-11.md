## 🔧 技术方案与实施

### 1. 数据扩充策略

**数据源:** `cnt_dataset_v4_real.csv` (533 行)

**缺失值处理:**
| 特征 | 缺失率 | 处理策略 |
|------|--------|----------|
| cvd_temperature_C | 81.4% | 中位数填充 + is_cvd 二元特征 |
| length_um | 48.6% | 中位数填充 |
| catalyst | 65% | has_catalyst 二元特征 |
| carbon_source | 65% | has_carbon_source 二元特征 |

**衍生特征工程:**
```python
# 1. 长径比 (捕捉几何效应)
aspect_ratio = length_um * 1000 / diameter_nm

# 2. 直径对数 (非线性效应)
log_diameter = log10(diameter_nm + 1e-6)

# 3. SWCNT 标识 (单壁 vs 多壁)
is_swcnn = (layers == 1)

# 4. CVD 工艺标识
is_cvd = (method == 'CVD')

# 5. 温度归一化
temp_normalized = cvd_temperature_C / 1000.0  # 非 CVD 填 0

# 6-7. 催化剂/碳源存在性
has_catalyst = catalyst.notna()
has_carbon_source = carbon_source.notna()

# 8. 体积分数估算
volume_fraction_est = 1.0 / (diameter_nm ** 2) * layers
```

**最终特征集 (11 个):**
- 核心 (3): diameter_nm, length_um, layers
- 衍生 (8): aspect_ratio, log_diameter, is_swcnn, is_cvd, temp_normalized, has_catalyst, has_carbon_source, volume_fraction_est

### 2. 模型训练

**模型:** Gaussian Process Regressor

**核函数:**
```
1.11² × RBF(length_scale=[3.21e+04, 217, 7.41, 0.0192, 1.13, 3.81e-05, 4.62e+03, 1e+05, 8.45e+03, 1.61e-05, 5e+04]) 
+ WhiteKernel(noise_level=0.158)
```

**数据集划分:**
- 训练集：205 样本 (75%)
- 测试集：69 样本 (25%)

**性能指标:**
- R² = 0.799
- MAE = 1.92e-01 S/m
- RMSE = 2.72e-01 S/m
- 交叉验证 R² = 0.68 ± 0.01

### 3. SHAP 可解释性分析

**特征重要性排名:**

| 排名 | 特征 | SHAP 值 | 相对重要性 | 物理解释 |
|------|------|---------|------------|----------|
| 1 | diameter_nm | 0.680 | 68% | 量子限域效应主导 |
| 2 | cvd_temperature_C | 0.270 | 27% | 结晶质量影响 |
| 3 | length_um | 0.120 | 12% | 电子传输路径长度 |
| 4 | layers | 0.101 | 10% | 并联导电通道 |

**物理机制解释:**

1. **直径主导效应 (68%)**
   - 小直径 CNT (<2nm) 表现出更强的量子限域效应
   - 电子态密度更集中，费米能级附近态密度更高
   - 金属性 CNT 比例增加，电导率提升

2. **温度关键作用 (27%)**
   - CVD 温度影响碳原子排列有序度
   - 高温 (800-1000°C) 促进石墨化，sp²杂化更完整
   - 缺陷密度降低，电子散射减少

3. **几何因素 (长度/层数，共~22%)**
   - 长径比影响电子平均自由程
   - 多层 CNT 提供并联导电路径
   - 接触电阻随层数增加而降低
